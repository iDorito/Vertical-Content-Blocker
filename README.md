## 🧠 The Science Behind the Block

This extension isn't just a filter; it's a structural defense mechanism for your brain.

* **Dopamine Loops:** By blocking vertical "infinite" feeds, we prevent the **Variable Ratio Reinforcement** schedule that mimics slot machine addiction. Every swipe is a pull of the lever. By removing the lever, we starve the addiction.
* **Attention Protection:** We eliminate the **Attentional Blink** caused by 15-second context switching. Rapidly shifting from a comedy sketch to a tragedy to a dance trend causes cognitive exhaustion, destroying your prefrontal cortex's ability to engage in deep, sustained work.
* **Friction as a Feature:** The built-in mathematical bypass acts as a **Cognitive Speed Bump**, forcing your brain to switch from System 1 (automatic, impulsive thinking) to System 2 (logical, deliberate thinking) before accessing restricted content.

## ⚙️ Manual Installation Guide

Until this extension is officially published on the web stores, you can install it manually in **Developer Mode** on any major browser.

### 📥 Step 1: Download the Extension
1. Download the beta release version in the `releases` section.
2. Extract the downloaded `.zip` file into a folder on your computer (e.g., `Documents/Neural-Shield`).

### 🌐 Step 2: Install by Browser

#### 🔴 Google Chrome / Brave / Vivaldi
1. Open your browser and navigate to `chrome://extensions/` (or `brave://extensions/`).
2. Toggle on **Developer mode** in the top-right corner.
3. Click the **Load unpacked** button that appears in the top left.
4. Select the `Neural-Shield` folder you extracted in Step 1.
5. *Done.* Pin the extension to your toolbar.

#### 🔵 Microsoft Edge
1. Open Edge and navigate to `edge://extensions/`.
2. Turn on **Developer mode** using the toggle on the left sidebar or top right.
3. Click the **Load unpacked** button.
4. Select your extracted `Neural-Shield` folder.
5. *Done.* #### 🦊 Mozilla Firefox
*Note: Firefox wipes manual extensions when you close the browser unless you sign it, but this is how to test it:*
1. Open Firefox and navigate to `about:debugging#/runtime/this-firefox`.
2. Click the **Load Temporary Add-on...** button.
3. Navigate to your extracted `Neural-Shield` folder and select the `manifest.json` file.
4. *Done.* (You will need to repeat this step if you restart Firefox, until the official add-on is published).

* **Audio Kill-Switch:** Forces an active loop that mutes and unloads media elements if you accidentally trigger a direct video link.
